About

Here are some school-related projects completed in C# using the MonoGame framework. The MonoGame framework was used as per course requirements.

The MonoGame (formerly XNA) framework is a free C# framework used to make games for multiple platforms. More information about the MonoGame framework can be found on Wikipedia , or on the MonoGame website.


PROG2370 - Course Project

[Github Repo Link]

About:

This game was made in MonoGame as the final project for the PROG2370: Game Programming with Data Structures course. The game I made was heavily inspired by the video game Tom Clancy's Rainbow Six: Seige.

My game shares similar features such as destroyable walls and multiple usable weapons. All models in game are created by me, and all code are my own unless otherwise stated.
